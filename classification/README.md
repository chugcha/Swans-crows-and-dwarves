**Директория ```classification``` содержит файлы, связанные с количественным анализом данных исследования.**

Файл **classification.xlsx** содержит таблицы с классификацией вступлений по нескольким параметрам.

Файл **age_stats.xlsx** содержит таблицы и диаграммы, посвящённые связи возраста респондентов с их ответами.
