**Директория graphs содержит две папки:**

1. В папке ```input``` содержатся файлы, благодаря которым можно запустить тетрадку Jupyter notebook и получить разные варианты графов со связями между словами в вступлениях:
   - ```graphs.ipynb```: тетрадка с кодом
   - ```all_data.csv```: все собранные данные
   - ```intro_cleaned.csv```: все вступления в алфавитном порядке, отсмотренные вручную
   - ```pairs_lemmatized_cleaned.csv```: лемматизированный список слов, отсмотренный вручную
3. В папке ```output``` содержаться готовые графы и списки вступлений:
   - ```intro_uncleaned.csv```: список всех вступлений
   - ```intro.csv```: список всех полученных вступлений, отсортированных в алфавитном порядке
   - ```pairs_lemmatized.csv```: лемматизированный список пар слов, встречающхся вместе, с частотностью
   - ```pairs_cleaned.csv```: список пар слов без стоп-стов, с частотностью
   - ```intro_network_all.html```: граф, постренный по всем парам слов
   - ```intro_network_2andmore.html```: граф по парам слов с ограничением частотности >= 2
   - ```intro_network_stop.html```: граф по парам слов с ограничением частотности и без стоп-слов
