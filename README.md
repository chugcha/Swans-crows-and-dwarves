# Летел лебедь... Или все-таки ворона?

Этот репозиторий содержит данные и материалы социолингвистического исследования детского фольклора, посвященного вариативности вступления к популярной детской игре, известной в классическом (московском) варианте как **«Летел лебедь по синему небу...»**.

## Об исследовании

Цель проекта: сбор, анализ и систематизация различных вариантов вступления к игре, их географическое распределение и связь с возрастом респондентов. Мы разработали и провели онлайн-опрос, отсортировали ответы, выделили частые образы. Также создали карты, графы связей для наглядного представления результатов.
